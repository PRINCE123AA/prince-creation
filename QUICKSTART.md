# Project Quick Start Guide

## Running the Backend

1. **Clone the Repository**:  
   Clone the repository to your local machine using the command:
   ```bash
   git clone https://github.com/PRINCE123AA/prince-creation.git
   ```

2. **Navigate to the Backend Folder**:  
   Change the directory to the backend folder:
   ```bash
   cd prince-creation/backend
   ```

3. **Install Dependencies**:  
   Install the necessary packages:
   ```bash
   npm install
   ```

4. **Run the Backend Server**:  
   Start the backend server on localhost:5000:
   ```bash
   npm start
   ```
   Your backend should now be running at [http://localhost:5000](http://localhost:5000)

## Running the Frontend

1. **Navigate to the Frontend Folder**:  
   Change the directory to the frontend folder:
   ```bash
   cd ../frontend
   ```

2. **Install Dependencies**:  
   Install the required packages:
   ```bash
   npm install
   ```

3. **Run the Frontend Server**:  
   Start the frontend server on localhost:5173:
   ```bash
   npm run dev
   ```
   Your frontend should now be accessible at [http://localhost:5173](http://localhost:5173)

## Accessing the Project

- Open your browser and navigate to [http://localhost:5173](http://localhost:5173) to see the application in action. 
- The backend server will handle requests on port 5000 and the frontend will be served from port 5173.  

## Troubleshooting

If you encounter any issues, please ensure that:
- All dependencies are correctly installed.
- Both servers are running without errors.
